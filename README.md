# Oncology Clinical Trial Analysis

## Overview

This project analyzes a synthetic oncology clinical trial dataset using **PostgreSQL** and **SQL**. The goal is to explore patient demographics, treatment effectiveness, survival outcomes, and adverse events while demonstrating practical SQL skills used in healthcare data analytics.

This project is being developed as a portfolio case study following a structured analytics workflowвЂ”from database design and data quality assessment to exploratory analysis and clinical insights.

---

## Project Objectives

* Design and implement a relational database in PostgreSQL.
* Import and validate a clinical trial dataset.
* Perform data quality assessment.
* Explore patient demographics and disease characteristics.
* Evaluate treatment effectiveness across different cancer types.
* Analyze progression-free survival (PFS) and overall survival (OS).
* Investigate adverse event severity and trial completion.
* Demonstrate SQL techniques used in real-world healthcare analytics.

## Dataset

This project uses a **synthetic oncology clinical trial dataset** containing **5,000 patient records**.

The processed dataset includes the following variables:

* Patient ID
* Cancer Type
* Cancer Stage
* Treatment Arm
* Age
* Sex
* ECOG Performance Score
* Smoking History
* Comorbidity Count
* Treatment Cycles
* Best Response
* Tumor Size Change (%)
* Progression-Free Survival (Weeks)
* Overall Survival (Weeks)
* Maximum Adverse Event Grade
* Trial Completion Status
