# Oncology Clinical Trial Analysis

## Overview

This project analyzes a synthetic oncology clinical trial dataset using PostgreSQL.

The analysis focuses on identifying trends in patient demographics, treatment effectiveness, survival outcomes, and adverse events through SQL.

---

## Objectives

- Explore patient demographics
- Analyze cancer stage distribution
- Evaluate treatment response
- Compare progression-free and overall survival
- Assess adverse event severity
- Investigate clinical trial completion rates

---

## Tech Stack

- PostgreSQL
- pgAdmin 4
- SQL
- GitHub

---

## Dataset

A synthetic dataset containing 5,000 oncology clinical trial records.
